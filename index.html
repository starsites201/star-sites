O que é o Termux?

Termux é um poderoso terminal para Android que permite rodar comandos Linux, programar e até criar servidores direto do celular.

Comandos Úteis

  • pkg update && pkg upgrade - Atualiza os pacotes
  • pkg install git - Instala o Git
  • pkg install python - Instala o Python
  • ls - Lista arquivos no diretório

Tutoriais Básicos

2. Como clonar um repositório do GitHub: git clone [link]

3. Como rodar scripts em Python: python script.py
